Forge Design System

Ember on forged steel.

The visual language for Forge — an engineering operating system that shapes raw specs into shipped software. Precise, industrial, warm where it matters. Deliberately not the cool indigo of the tools it replaces.

01

The mark

A geometric monogram “F” that reads as a struck anvil — three heavy ember strokes standing on a steel plinth, with a bright spark flying off the corner: the instant the hammer meets hot metal. One motif, anvil + spark — never a second metaphor.

03

Type

A geometric display voice for headings (engineered, tight), a clean system sans for UI, and a real mono for code, IDs and keys. Display is self-hosted Bricolage Grotesque (distinctive, offline via @fontsource); code uses JetBrains Mono; body stays a clean system sans — zero CDN font dependency.

06

Principles

Ember is precious.
One primary action per view. If everything is ember, nothing is.
Steel does the work.
Structure, surfaces and text are cool graphite; ember only marks intent.
Spark = focus.
The bright spark tone is reserved for focus rings and the logo — focus always reads as “struck”.
Warm hovers.
Surfaces heat toward amber under the cursor, never cooler. The interface warms as you work.
